# Dennisfx Deriv Dashboard

Personal third-party Deriv trading dashboard using Next.js and Deriv WebSocket API.

- App ID: 126398
- Features: Live ticks (Volatility 100), balance, open positions
- Domain: https://dennisfx.vercel.app

## Setup
1. npm install
2. Create .env.local with DERIV_API_TOKEN=your-token-here
3. npm run dev
